# Opportunity Intake Folder

Use this folder for temporary opportunity-document intake when preparing capture alignment.

## Rules
- Public solicitation documents may be placed here temporarily for Codex analysis.
- Proprietary, confidential, export-controlled, or sensitive files should not be committed.
- If this repository is public/client-visible, keep opportunity files local-only.

## Codex behavior
- Analysis output should be written to `docs/opportunity-alignment.md`.
- Public site updates must not disclose confidential solicitation details.

## Commit safety
- This repo is configured to ignore files in `capture/opportunity/*` by default.
- `README.md` and `.gitkeep` remain tracked so the folder structure is preserved.
