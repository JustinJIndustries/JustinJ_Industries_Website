# Capture Workspace

This folder is for temporary government-opportunity intake and analysis support.

## Usage
- Place public solicitation documents here temporarily when you want Codex to analyze an opportunity.
- Do not commit proprietary, confidential, controlled, or sensitive files.
- If this repository is public (or client-visible), opportunity files should remain local-only and uncommitted.

## Output policy
- Codex should produce analysis outputs in: `docs/opportunity-alignment.md`
- Public website updates must not reveal confidential solicitation details, proprietary teaming information, or restricted procurement content.

## Structure
- `capture/opportunity/` is the drop zone for temporary opportunity documents.
- `capture/opportunity/README.md` explains handling and commit safety rules.
